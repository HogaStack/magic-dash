# `magic-dash-pro`配置参数

`magic-dash-pro`的配置集中在`configs/`目录。它继承多页面、持续扩展模板的配置方式，并额外增加面向复杂用户与部门关系的鉴权、会话、安全和数据库配置。

## `configs/base_config.py`

| 参数 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `app_title` | `str` | `"Magic Dash Pro"` | 应用标题 |
| `app_version` | `str` | `"dev"` | 应用版本；通过`CLI`创建项目时会被替换为当前`magic-dash`版本 |
| `enable_version_changelog_modal` | `bool` | `False` | 是否启用版本更新日志通知弹窗 |
| `version_changelog_markdown_folder` | `str` | `"changelogs"` | 更新日志`Markdown`文件目录 |
| `app_secret_key` | `str` | `Flask`：`"magic-dash-pro-demo"`；`FastAPI`：`"magic-dash-pro-fastapi-demo-secret-key"` | 应用密钥，生产环境必须替换；`FastAPI`后端会基于此密钥签发`JWT`，建议长度不低于32字节 |
| `app_session_cookie_name` | `str` | `Flask`：`"magic_dash_pro_session"`；`FastAPI`：`"magic_dash_pro_fastapi_session"` | 应用会话`cookie`名称 |
| `login_session_expire_seconds` | `int` | `12 * 60 * 60` | `FastAPI`后端专用，登录会话有效期，单位为秒 |
| `remember_login_expire_seconds` | `int` | `30 * 24 * 60 * 60` | `FastAPI`后端专用，勾选“记住我”后的登录会话有效期，单位为秒 |
| `min_browser_versions` | `List[dict]` | `Chrome`88、`Firefox`78、`Edge`100 | 浏览器最低版本限制规则 |
| `strict_browser_type_check` | `bool` | `False` | 是否拦截不在规则内的浏览器类型 |
| `enable_duplicate_login_check` | `bool` | `False` | 是否启用重复登录辅助检查 |
| `duplicate_login_check_interval` | `Union[int, float]` | `10` | 重复登录检查轮询间隔，单位为秒 |
| `session_token_cookie_name` | `str` | `"session_token"` | 登录会话`token`对应的`cookie`名称 |
| `enable_fullscreen_watermark` | `bool` | `False` | 是否启用全屏水印 |
| `fullscreen_watermark_generator` | `Callable` | 当前用户名 | 全屏水印内容生成函数 |
| `enable_login_rsa_crypto` | `bool` | `False` | 是否启用登录密码`RSA`加密传输 |
| `rsa_public_key_path` | `str` | `"magic_dash_pro_public_key.pem"` | `RSA`公钥文件路径 |
| `rsa_private_key_path` | `str` | `"magic_dash_pro_private_key.pem"` | `RSA`私钥文件路径 |
| `enable_login_captcha` | `bool` | `False` | 是否启用登录页滑块验证 |

## `configs/layout_config.py`

| 参数 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `core_side_width` | `int` | `350` | 核心页面侧边栏宽度，单位为像素 |
| `login_content_type` | `"image" \| "video" \| "gradient"` | `"gradient"` | 登录页面内容区域渲染形式：图片内容、视频内容或渐变色背景 |
| `login_page_layout` | `"content-left" \| "form-left" \| "centered"` | `"centered"` | 登录页面布局形式：左侧内容右侧登录控件、左侧登录控件右侧内容或居中登录控件 |
| `core_layout_type` | `"single" \| "tabs"` | `"single"` | 核心页面展示方式：单页面或多标签页 |
| `show_core_page_search` | `bool` | `True` | 是否在页首显示页面搜索框 |

## `configs/router_config.py`

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| `index_pathname` | `str` | 应用首页对应的`pathname` |
| `core_side_menu` | `List[dict]` | 侧边菜单完整结构 |
| `wildcard_patterns` | `dict` | 通配页面正则规则 |
| `valid_pathnames` | `dict` | 有效页面`pathname`到页面标题的映射 |
| `independent_core_pathnames` | `List[Union[str, re.Pattern]]` | 不嵌入核心布局、独立渲染的核心页面 |
| `public_pathnames` | `List[str]` | 无需登录和权限校验的公开页面 |
| `side_menu_open_keys` | `dict` | 指定`pathname`访问时默认展开的侧边菜单层级 |

## `configs/auth_config.py`

| 参数 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `roles` | `dict` | `admin`、`normal` | 系统角色定义 |
| `normal_role` | `str` | `"normal"` | 常规用户角色标识 |
| `admin_role` | `str` | `"admin"` | 管理员角色标识 |
| `pathname_access_rules` | `dict` | `admin`全部可访问，`normal`排除登录日志 | 不同角色对页面菜单`key`的访问规则 |

`pathname_access_rules`支持三种规则：

| 类型 | 说明 |
| --- | --- |
| `all` | 可访问全部页面 |
| `include` | 仅可访问`keys`中声明的页面；首页会自动纳入 |
| `exclude` | 可访问除`keys`外的页面 |

## `configs/database_config.py`

| 参数 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `database_type` | `"sqlite" \| "postgresql" \| "mysql"` | `"sqlite"` | 应用数据库类型 |
| `postgresql_config` | `dict` | 本地`PostgreSQL`示例连接 | `PostgreSQL`连接参数 |
| `mysql_config` | `dict` | 本地`MySQL`示例连接 | `MySQL`连接参数 |

使用`PostgreSQL`时需额外安装：

```bash
pip install psycopg2-binary
```

使用`MySQL`时需额外安装：

```bash
pip install pymysql
```

## 安全建议

- 生产环境务必替换`app_secret_key`。
- 多套应用部署在同一域名或主机地址下时，为每套应用设置不同的`app_session_cookie_name`和`session_token_cookie_name`。
- 开启`enable_login_rsa_crypto`时**必须**配合`HTTPS`环境使用；非`HTTPS`环境不应启用该配置。
- 初始化后的默认管理员密码应尽快修改。
