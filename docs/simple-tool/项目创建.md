# `simple-tool`项目创建

`simple-tool`是最轻量的内置模板，适合快速制作单页面数据处理工具、计算器、转换器、报表生成入口或内部自动化小工具。

## 创建项目

```bash
magic-dash create --name simple-tool
```

命令会先提示选择后端类型，再提示输入项目名称。直接回车时，默认生成`simple-tool`目录。

后端类型可选择：

- `Flask`：默认后端
- `FastAPI`：生成后会将`dash.Dash()`实例切换为`backend="fastapi"`，并在`requirements.txt`中补充`dash[fastapi]`、`fastapi`、`uvicorn`等依赖

指定项目生成父目录：

```bash
magic-dash create --name simple-tool --path ./workspace
```

直接指定后端类型：

```bash
magic-dash create --name simple-tool --backend flask --path ./workspace
magic-dash create --name simple-tool --backend fastapi --path ./workspace
```

最终目录为：

```text
./workspace/项目名称
```

## 安装依赖并启动

```bash
cd simple-tool
pip install -r requirements.txt
python app.py
```

默认访问地址：

```text
http://127.0.0.1:8050
```

## 初始项目结构

```text
simple-tool/
├── app.py
└── requirements.txt
```

`app.py`同时包含应用实例、页面布局和示例回调；`requirements.txt`记录运行该单页工具所需依赖。

## 适合从这里开始的场景

- 输入少量参数，点击按钮后输出计算结果
- 封装一个脚本能力为`Web`表单
- 构建只面向少量内部用户的轻量[Dash](https://github.com/plotly/dash)工具
- 在正式拆分多页面应用前快速验证交互流程
